CSS usage

Both compressed and uncompressed CSS files are included for Startup in css/ path. Using the minified(startup.min.css) file will result in better performance.This also applies for all themes.

  1. css/startup.css
  2. css/startup.min.css

You can link Startup CSS to HTML page as given below.

<link rel="stylesheet" type="text/css" href="css/startup.min.css">

<!-- If you prefer some Startup theme -->
<link rel="stylesheet" type="text/css" href="css/startup-sassy.min.css">

JS usage

JS file is included for Startup in js/ path. startup.js should be included after all its dependencies.

  1. js/startup.js

You can link Startup JS to HTML page as given below.

<script type="text/javascript" src="js/startup.js"></script>

LESS usage

All the Startup specific LESS files are kept in less/startup/ path including themes.

  1. less/startup/variables.less
  2. less/startup/navbar.less
  3. less/startup/hero.less
  4. less/startup/skills.less
  5. less/startup/services.less
  6. less/startup/portfolio.less
  7. less/startup/timeline.less
  8. less/startup/opensource.less
  9. less/startup/contact-me.less
  10. less/startup/footer.less
  11. less/startup/global.less
  12. less/startup/startup.less

If you want to use the LESS version of Startup in your application copy the less/startup/ into your application except themes folder. And import the less/startup/startup.less in your application less file. 

@import "/path-to-startup/startup.less";

// If you prefer to use some Startup theme, replace startup.less with theme file.
// For example to import the startup-sassy.less theme you need to replace startup.less with startup-sassy.less

@import "/path-to-startup/startup-sassy.less";
// NOTE: The import statements in the theme should point to the correct path of the LESS files.

Generating custom theme in Startup

If you want to generate your own theme, just change one less variable in main importer file.You need to change the @theme-bg in the main Startup LESS importer. If you want to change the theme skin color to #d43631 follow the below code snippet.

// Content of the main LESS importer(startup.less) will be

@theme-bg:        #d43631; // This is your required theme color. Thats it.
@theme-color:     #fff; // Change of text color is not recommended.

@import "variables";
@import "navbar";
@import "hero";
@import "skills";
@import "services";
@import "portfolio";
@import "timeline";
@import "opensource";
@import "contact-me";
@import "footer";
@import "global";

Recommended image sizes

Using recommended image size will avoid image scaling at browser which will increase the page loading speed.

Image type Image Size
Startup logo 100x50
Freelancer profile image 425x585
Service image 360x220
Project image 360x250
More project screenshots (In modal) 568x355
Client image (In testimonial) 125x125
Event image (In timeline) 455x316
Opensource project logo 100x100
