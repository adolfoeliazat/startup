CSS usage

Both compressed and uncompressed CSS files are included for Solo in css/ path. Using the minified(solo.min.css) file will result in better performance.This also applies for all themes.

  1. css/solo.css
  2. css/solo.min.css

You can link Solo CSS to HTML page as given below.

<link rel="stylesheet" type="text/css" href="css/solo.min.css">

<!-- If you prefer some Solo theme -->
<link rel="stylesheet" type="text/css" href="css/solo-sassy.min.css">

JS usage

JS file is included for Solo in js/ path. startup.js should be included after all its dependencies.

  1. js/startup.js

You can link Solo JS to HTML page as given below.

<script type="text/javascript" src="js/startup.js"></script>

LESS usage

All the Solo specific LESS files are kept in less/solo/ path including themes.

  1. less/solo/variables.less
  2. less/solo/navbar.less
  3. less/solo/hero.less
  4. less/solo/skills.less
  5. less/solo/services.less
  6. less/solo/portfolio.less
  7. less/solo/timeline.less
  8. less/solo/opensource.less
  9. less/solo/contact-me.less
  10. less/solo/footer.less
  11. less/solo/global.less
  12. less/solo/solo.less

If you want to use the LESS version of Solo in your application copy the less/solo/ into your application except themes folder. And import the less/solo/solo.less in your application less file. 

@import "/path-to-solo/solo.less";

// If you prefer to use some Solo theme, replace solo.less with theme file.
// For example to import the solo-sassy.less theme you need to replace solo.less with solo-sassy.less

@import "/path-to-solo/solo-sassy.less";
// NOTE: The import statements in the theme should point to the correct path of the LESS files.

Generating custom theme in Solo

If you want to generate your own theme, just change one less variable in main importer file.You need to change the @theme-bg in the main Solo LESS importer. If you want to change the theme skin color to #d43631 follow the below code snippet.

// Content of the main LESS importer(solo.less) will be

@theme-bg:        #d43631; // This is your required theme color. Thats it.
@theme-color:     #fff; // Change of text color is not recommended.

@import "variables";
@import "navbar";
@import "hero";
@import "skills";
@import "services";
@import "portfolio";
@import "timeline";
@import "opensource";
@import "contact-me";
@import "footer";
@import "global";

Recommended image sizes

Using recommended image size will avoid image scaling at browser which will increase the page loading speed.

Image type Image Size
Solo logo 100x50
Freelancer profile image 425x585
Service image 360x220
Project image 360x250
More project screenshots (In modal) 568x355
Client image (In testimonial) 125x125
Event image (In timeline) 455x316
Opensource project logo 100x100
